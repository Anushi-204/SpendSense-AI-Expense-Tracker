# SpendSense – AI Expense Tracker

## Overview
SpendSense is an AI-powered expense tracking web application built using Python Flask. It helps users manage expenses, analyze spending patterns, and get AI-based budget predictions.

## Features
- User Login & Registration
- Add/Edit/Delete Expenses
- Expense Category Tracking
- Monthly Reports
- Interactive Charts
- AI Expense Prediction
- Budget Suggestions

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js
- Pandas
- Scikit-learn

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open:
```bash
http://127.0.0.1:5000
```

- Improves financial tracking
